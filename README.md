# 说明

使用 vue3 + pinia + vue-router 开发的民宿类 h5 应用，构建工具使用 Vite。学习 coderwhy 前端系统课的vue3阶段的副产品（有机会给coderwhy补张门票）。

实现的关键页面：
- 首页
  - 日期选择
  - 房屋列表，触底加载
- 房屋详情页
  - 各个section的布局
  - 集成百度地图
